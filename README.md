# Villager Override
![banner](https://cdn.modrinth.com/data/cached_images/ec9a7a640f3f3550072d61f1b40e95214f6d807d.png)

Allows you to override the biome skins of villagers. For example make every villager looks like a jungle villager.

### Supported Platforms
Villager Override currently support client-side for both fabric and neoforge. It has only been tested on 1.21.11, but it should work on everything after 1.21.6.

## Configuration
The biome to override with can be configured by finding the config in your mod loaders mod list.
